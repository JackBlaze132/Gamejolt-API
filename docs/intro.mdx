# 🕹️ Game Jolt API Wiki

Welcome to the **Modern Interactive Documentation** for the Game Jolt API. This wiki is designed to help you integrate social and progress-based features into your games with ease, focusing on **Unity C#** and **REST transitions**.

## 🚀 Key Features

*   **⚡ Interactive Emulators**: Test API logic directly in the browser before writing a single line of code.
*   **🛠️ Unity-First Patterns**: Ready-to-use C# snippets for Trophies, Scores, and Data Storage.
*   **📦 Batch Optimization**: Learn how to bundle network requests to keep your game running smoothly.
*   **🎨 Retro-Modern Aesthetic**: Pixel-perfect readability with high-contrast neon themes.

## 🏁 Quick Start

To begin integrating Game Jolt into your project, follow these core steps:

1.  **[Authentication](authentication.md)**: Connect your players to the Game Jolt ecosystem.
2.  **[User Operations](user-operations.md)**: Fetch profiles and manage friend lists.
3.  **[Trophies & Achievements](unity-achievements.md)**: reward players for their milestones.
4.  **[Leaderboards](scores.md)**: Track high scores and create competitive challenges.
