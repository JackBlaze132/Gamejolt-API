# 📝 Overview

Game Jolt allows you to add trophies to your games! Trophies come in four materials: bronze, silver, gold, and platinum. This is to reflect how difficult it is to achieve a trophy.

## Materials
*   **Bronze**: Should be easy to achieve.
*   **Silver**: Medium difficulty.
*   **Gold**: Hard to achieve.
*   **Platinum**: Very hard to achieve.

On Game Jolt, trophies are always listed in order from easiest to most difficult to achieve.

## Secret Trophies
You can tag trophies as "secret". A secret trophy's image and description is not visible until a gamer has achieved it.

## API Endpoints
*   [Fetch](./fetch.md) - Fetches trophies with various attributes.
*   [Add Achieved](./add-achieved.md) - Sets a trophy as achieved for a user.
*   [Remove Achieved](./remove-achieved.md) - Removes an achieved trophy.

## Remarks
*   You can manage trophies for your game from your game's dashboard.
